# 第一部分：岗位理解框架（一张图）

```
                    AI内容运营 = TOB内容推广

┌─────────────────────────────────────────────────────┐
│                    ① 深度使用产品                      │
│  自己先用无限画布做素材 → 踩坑 → 发现技巧 → 积累案例    │
│  不自己用 = 没东西可写 = 内容没有说服力                  │
├─────────────────────────────────────────────────────┤
│                    ② 内容创作                         │
│  把使用过程变成：教程 / 案例 / 对比 / 干货 / 客户故事    │
│  核心原则：不是打广告，是展示"我用这个工具做了什么效果"    │
├─────────────────────────────────────────────────────┤
│                    ③ 多平台分发                       │
│  公众号(深度) + 小红书(引流) + 短视频(展示) + 朋友圈(触达) │
│  每个平台内容形式不同，但核心信息一致：这个AI工具能帮你省钱  │
├─────────────────────────────────────────────────────┤
│                    ④ 客户跟进                         │
│  内容引来咨询 → 演示 → 试用 → 收集反馈 → 优化内容        │
│  客户的问题 = 下一篇内容的选题                          │
│  客户的成功 = 下一个案例的素材                          │
└─────────────────────────────────────────────────────┘

                    形成闭环 ↑
  用产品→做内容→引客户→收反馈→优化内容→引更多客户→……
```

# 第二部分：自我介绍的逻辑线

**不是背稿，是按这条线讲：**

```
我是谁(10秒) → 我做过什么(30秒) → 为什么适合这个岗位(40秒) → 我怎么理解这个岗位(30秒) → 收尾(10秒)
```

**逐句拆开：**

| 段落 | 讲什么 | 时长 |
|------|--------|------|
| 开头 | 基本身份 + 一个记忆点 | 10秒 |
| 项目 | 菠萝烤盐 + 小红书，各一句话 | 30秒 |
| 匹配 | 三条：AI能力 + 内容经验 + 沟通跟进 | 40秒 |
| 认知 | 我对岗位的理解（四个环节） | 30秒 |
| 收尾 | 表达意愿 | 10秒 |

# 第三部分：逐字稿

## 版本A：标准版（约2分钟，通用）

> 面试官你好，我叫罗楚婷，广东财经大学汉语言文学专业2025届毕业生。
>
> 过去一年我主要做了两件事。
>
> 第一件，我独立做了一款叫「菠萝烤盐」的考研择校工具。从需求调研到产品设计到开发上线，全程用AI辅助完成。这个产品在小红书获得了14万曝光、3000多用户，在「汉语言文学考研」和「考研择校」两个板块都冲到过热度第一，而且零推广成本。
>
> 第二件，我运营了一个小红书个人账号。不做垂直赛道，纯靠内容感觉和AI辅助创作，做到了1500多粉丝和多条爆款笔记。
>
> 为什么我觉得自己适合这个岗位？三个原因。
>
> 第一，我是AI的重度使用者。不只是"用过"，是用AI做出了真实产品。你们推无限画布给电商客户——我自己就是用AI从零到一的人，我理解一个AI产品从学习到上手到做出成果的完整过程。这让我能更快吃透产品，做出有说服力的案例。
>
> 第二，我做过内容驱动的增长。我的产品零推广成本获客，靠的就是在小红书持续输出对用户有价值的内容。这套逻辑搬到TOB场景是一样的——不发广告，发客户看完觉得有用的东西。
>
> 第三，我能跟人打交道。一年教培经历，面对家长和学生，满意度100%，续课率超机构平均25%，零投诉。我知道怎么把复杂的东西讲简单，也知道客户的问题要跟进到底。
>
> 我理解这个岗位的核心是四个环节：先自己深度使用产品做出案例，然后把案例变成不同平台的内容去吸引潜在客户，客户来了做演示和跟进，最后把客户的反馈和成功案例再变成新的内容。这是一个从使用产品到内容创作到客户服务的完整闭环。我过去做的每一个项目，本质上都在跑这个闭环。
>
> 我今天带了电脑，可以展示我的作品，也期待了解无限画布这个产品。

## 版本B：针对"无限画布"定制版（如果面试官聊得比较技术向）

> 面试官好，我是罗楚婷，广财2025届，汉语言文学专业。
>
> 我简单介绍一下自己。过去一年我用AI做了一款考研择校产品，3000多用户、14万曝光，从产品设计到内容推广全部独立完成。同时做小红书账号，1500多粉丝，纯靠内容跑自然流量。
>
> 来之前我了解了一下无限画布这个产品——它是帮电商商家做投流素材的AI工具，核心是降本提效。以前商家剪一条素材要几小时、十几块钱，现在10分钟、2块钱。
>
> 我觉得这个岗位最核心的能力是：你得自己先用产品用到能闭眼操作，然后把你用产品的过程——怎么做的、效果怎么样、跟以前比省了多少——变成不同平台的内容。公众号写深度教程、小红书发实操对比、短视频展示效率差异、朋友圈做日常触达。内容不是打广告，是让电商老板看完之后自己想试。
>
> 然后客户来了，你要演示、跟进、收集反馈。客户提的问题就是下一篇内容的选题，用得好就是下一个案例的素材。这就形成了一个循环。
>
> 我过去一年做的事情，跟这个循环一模一样——自己做产品、做内容、做用户运营。区别只是我是C端考研产品，你们是B端电商AI产品。但"把产品价值翻译成内容"这件事，底层逻辑完全一样。
>
> 我带电脑了，可以展示我的作品，也想听听你们目前对无限画布的内容运营是怎么规划的。

# 第四部分：面试节奏建议

```
0:00 - 0:15  自我介绍 + 项目亮点
0:15 - 0:45  为什么匹配（三条：AI+内容+沟通）
0:45 - 1:15  我对岗位的理解（四个环节的闭环）
1:15 - 1:20  收尾 + 带电脑可展示
1:20 - 2:00  面试官提问（灵活应对）
2:00 - 2:15  你反问面试官
```

**关键提醒**：
- 不要背稿。把框架记住，自然地说
- 眼神交流比稿子重要
- 讲到"3000用户""14万曝光""热度第一"时停顿一下，让数字打进去
- 如果紧张就说慢一点，没人会觉得你说得慢有问题
